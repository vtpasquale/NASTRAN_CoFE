CORD2R

Defines a rectangular coordinate system using the coordinates of three points.

Format

123456789
CORD2RCIDRIDA1A2A3B1B2B3
C1C2C3

Example

CORD2R,18,,0.000000,0.0,0.0,0.0,-0.122,1.0, ,0.809,-0.588,0.122

Fields

Field Contents
CID Coordinate system identification number. (Integer > 0)
RID Not used.
Ai, Bi, Ci Coordinates of three points in basic coordinate system.

Remarks:

  1. The three points [(A1, A2, A3), (B1, B2, B3), (C1, C2, C3)] must be unique and noncolinear. The first point defines the origin. The second defines the direction of the z-axis. The third point defines a vector which, with the z-axis, defines the x-z plane. The reference coordinate system must be independently defined.
  2. The location of a grid point (P in the Fig.~\ref{f:CORD2R_Definition}) in this coordinate system is given by (X, Y, Z). \myfig{CORD2R_Definition}{CORD2R Definition.\cite{msc2011msc}}
  3. This entry is referenced exclusively by \texttt{GRID} entries, so that they can be specified in an alternative coordinate system. No other inputs or outputs will use an alternative (not basic) coordinate system.
