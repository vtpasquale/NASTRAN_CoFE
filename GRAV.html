GRAV

Defines acceleration vectors for gravity or other acceleration loading.

Format

123456789
GRAVSIDCIDAN1N2N3MB

Example

GRAV,9,0,29420.4,0.138814,0.0,-0.990318

Fields

Field Contents
SID Load set identification number. (Integer > 0)
CID Not used.
A Acceleration vector scale factor. (Real)
Ni Acceleration vector components measured in default coordinate system. (Real)
MB Not used.
