# SUBCASE
Delimits and identifies a subcase.

## Format
`SUBCASE=n`

## Examples
`SUBCASE=10`

## Describers
| Describer  | Meaning  |
| ---------- | -------- |
| n          | Subcase identification number. |

## Remarks
1. The equal sign can be replaced with a space (e.g., `SUBCASE 10`).
