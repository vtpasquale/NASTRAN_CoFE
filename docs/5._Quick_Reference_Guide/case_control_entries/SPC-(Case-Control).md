# SPC
Selects a single-point constraint set to be applied.

## Format
`SPC=n`

## Examples
`SPC=5`

## Describers
| Describer  | Meaning  |
| ---------- | -------- |
| n          | Set identification number of single-point constraint bulk data entries. |

## Remarks
1. The SPC set defined for the first subcase is used for all subcases.
