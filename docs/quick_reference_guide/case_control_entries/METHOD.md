Selects eigenvalue extraction parameters.

## Format
`METHOD(LHSD)=n`

## Examples
`METHOD=101`

## Describers
| Describer  | Meaning  |
| ---------- | -------- |
| LHSD       | Left hand side describers. These are ignored. |
| n          | Set identification number of an EIGRL bulk data entry. |
