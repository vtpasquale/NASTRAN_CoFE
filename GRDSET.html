GRDSET

Defines a static concentrated force at a grid point by specifying a vector.

Format

123456789
GRDSETCPCDPSSEID

Example

$1------2-------3-------4-------5-------6-------7-------8-------9-------

$GRDSET                                                 PS

GRDSET                                                  2456

Fields

Field Contents
CP Not used.
CD Not used.
PS Default permanent single-point constraints on all grid points. (Any combination of Integers 1 through 6 with no embedded blanks, or blank.)
SEID Not used.

Remarks:

  1. This entry is optional. When omitted, the default is no single-point constraints.
  2. The contents of field 8 of this entry are assumed for the corresponding field 8 of any GRID entry whose field 8 is blank. If field 8 on the GRID entry is blank, the default option is used for that field. If no permanent single-point constraints are desired, the default may be overridden on the GRID entry by making field 8 zero (rather than blank). Only one GRDSET entry may appear in the Bulk Data Section.
  3. The primary purpose of this entry is to minimize the burden of preparing data for problems with a large amount of repetition (e.g., two-dimensional pinned-joint problems).
