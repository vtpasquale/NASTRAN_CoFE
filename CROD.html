CROD

Defines a tension-compression-torsion element.

Format

123456789
CRODEIDPIDG1G2

Example

CROD,7,701,5,4

Fields

Field Contents
EID Element identification number. (1 < Integer < 100,000,000)
PID Property identification number of a PROD entry. (Integer > 0)
G1, G2 Grid point identification numbers of connection points. (Integer > 0; G1 ≠ G2 )

Remarks:

  1. Differential stiffness of the torsional component of stiffness is not captured for buckling eigenvalue problems.
